# Cookie Pool Example

This is a stripped-down example version of the Weibo Search Cookie Pool system.

## Installation

1. Clone the repository:
   ```bash
   git clone git@github-work:16yun-cn/cookie-pool-example.git
   cd cookie-pool-example
   ```

2. Install dependencies:
   ```bash
   pip install -e .
   ```

3. Set up environment variables:
   ```bash
   cp .env.example .env
   # Edit .env with your configuration
   ```
